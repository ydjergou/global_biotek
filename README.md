🌎Global Biotek

Global Biotek est une plateforme web développée avec Flask permettant la collecte, l'intégration et l'analyse des données de résistance aux antimicrobiens (AMR) dans les pays de la CEDEAO.

Le projet permet de :
* Analyser les données Biologiques
* Intégrer la base CARD
* Nettoyer les données

Technologies utilisées
* Python
* Flask
* HTML

 Installation du projet

1.  Cloner le repository

- git bash
git clone https://github.com/ydjergou/global_biotek.git
cd global_biotek

2.  Créer un environnement virtuel

Sur Windows
- Terminal
python -m venv venv
venv\Scripts\activate

3.  Installer les dépendances
- Terminal
pip install -r requirements.txt

 4. Lancer le projet
- Terminal
python app.py

🌐 Accéder à l'application

Ouvrir dans le navigateur :
http://127.0.0.1:5000
